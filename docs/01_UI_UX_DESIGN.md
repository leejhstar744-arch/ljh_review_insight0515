# UI/UX Design

## 디자인 방향
Apple 스타일 기반의 미니멀 UI

## 핵심 원칙
- 넓은 whitespace
- 단일 Accent Color
- 사진 및 콘텐츠 중심 레이아웃
- 최소한의 UI

## 컬러 시스템
```css
--color-primary: #0066cc;
--color-bg: #ffffff;
--color-soft: #f5f5f7;
--color-text: #1d1d1f;
```

## 주요 컴포넌트
- Hero Section
- Textarea
- Analyze Button
- Result Modal

## 반응형 전략
- Desktop
- Tablet
- Mobile
