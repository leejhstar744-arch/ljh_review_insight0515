# Project Overview

## 프로젝트명
AI Sentiment Analyzer

## 프로젝트 목적
사용자가 입력한 텍스트의 감정을 AI가 분석하여 긍정/부정/중립 결과와 신뢰도를 제공하는 웹 서비스 구축

## 핵심 기능
- 텍스트 입력
- 감성 분석
- OpenAI API 연동
- 결과 모달 출력
- 분석 기록 저장

## 기술 스택
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- AI: OpenAI API
- Database: Supabase
- Deployment: Vercel

## 제외 범위
- 로그인 기능
- 음성 분석
- 이미지 분석
- 관리자 페이지
